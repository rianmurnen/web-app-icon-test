Web App Icon Test

Setup

In the <head>:

	<link rel="manifest" href="/manifest.json">

Web App Manifest:


{
	"src": "/app-icon.svg",
	"sizes": "1024x1024",
	"type": "image/svg+xml",
	"purpose": "maskable"
}

The Icon

The SVG includes a prefers-color-scheme declaration.

Questions
